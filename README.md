# Employee Salary System

## Description
The Employee Salary System is a Python mini project developed using Object-Oriented Programming (OOP). It manages employee salary details, calculates manager salaries by including bonuses, reads employee data from a CSV file, reads bonus details from a JSON file, and generates a salary summary.

## Features
- Add employee details
- Calculate employee salary
- Calculate manager salary with bonus
- Read employee data from a CSV file
- Read bonus data from a JSON file
- Display employee salary status (Normal/High Income)
- Demonstrates inheritance and method overriding

## Technologies Used
- Python
- CSV Module
- JSON Module
- Object-Oriented Programming (OOP)
